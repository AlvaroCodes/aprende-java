# Ejercicio: AnalisisPelicula

Declara una variable `descripcion` (String) con formato "Pelicula: <título> | Año: <aaaa> | Director: <nombre> | Genero: <género>". 


```java
String descripcion = "Pelicula: Matrix | Año: 1999 | Director: Wachowski | Genero: Ciencia ficcion";
```


Muestra: la longitud del texto, el primer y último carácter, el `Título`, el `Año`, el `Director` y el `Género`. Además imprime "La película es de ciencia ficción" si contiene "Ciencia", "Película clásica" si contiene "1999", y finalmente la descripción completa en mayúsculas.
